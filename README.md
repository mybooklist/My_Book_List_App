# 📖 My Book List
Projeto desenvolvido como estudo de caso para o Trabalho de Conclusão de Curso (TCC) em Sistemas de Informação no IFBA - Campus Vitória da Conquista. O objetivo principal é investigar como a Inteligência Artificial (LLMs) pode atuar na avaliação de usabilidade em interfaces móveis.

# 📑 Sobre o Projeto
O My Book List é um aplicativo experimental focado na organização e gerenciamento de listas de leituras pessoais. Ele serve como laboratório para testar a eficácia de modelos de IA (Claude, Gemini e ChatGPT) na identificação de problemas de design e usabilidade.

## Principais Funcionalidades

- **Splash Screen:** Tela de carregamento inicial

- **Home:** Listagem de livros com filtros por categoria (Lido, Lendo, Quero Ler) e campo de busca por título ou autor

- **Detalhes do Livro:** Visualização de informações completas como gênero, avaliação, resumo e datas de leitura

- **Formulário de Cadastro:** Fluxo para adicionar novos títulos à biblioteca pessoal

- **Ações de Gerenciamento:** Opções para editar, excluir e compartilhar livros

# 🛠️ Tecnologias Utilizadas

- Framework: Flutter (Dart)
- Design/Prototipação: Figma
- Versionamento: GitHub

# 📝 Metodologia de Avaliação (UI/UX)
O projeto utilizou uma abordagem baseada em Heurísticas de Usabilidade. Foram selecionadas 4 das 10 heurísticas de Jakob Nielsen para a avaliação via IA:

- **H2 - Correspondência com o Mundo Real:** Linguagem e lógica familiar ao usuário.

- **H3 - Controle e Liberdade do Usuário:** Facilidade em desfazer ações ou sair de processos indesejados.

- **H4 - Consistência e Padrões:** Seguimento de convenções da plataforma para evitar confusão.

- **H8 - Design Estético e Minimalista:** Foco no essencial para reduzir a carga cognitiva.
